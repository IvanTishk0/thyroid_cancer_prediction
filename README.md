# thyroid_cancer_prediction
 
 Проект для предобработки клинических данных и построения моделей машинного обучения для прогнозирования рецидива (на основе датасета по раку щитовидной железы).
 
 ## Что внутри
 
 - **Предобработка**
   - очистка и приведение некоторых полей к числовому виду (например, длительность анамнеза)
   - one-hot encoding для категориальных признаков
   - обработка выбросов (IQR)
   - заполнение пропусков в числовых столбцах (`SimpleImputer(strategy='mean')`)
 - **Моделирование**
   - baseline/подбор гиперпараметров (в ноутбуке)
   - оценка качества по метрикам классификации
 - **Сравнительная таблица метрик**
   - `Sensitivity`, `Specificity`, `Precision`, `Accuracy`, `F1-score`, `ROC AUC`
 
 ## Структура репозитория
 
 - **`sample.ipynb`** — основной ноутбук с предобработкой, обучением и оценкой моделей
 - **`data/dataset.xlsx`** — исходные данные (Excel)
 - **`models/`** — каталог под сохранённые модели (если будешь добавлять сохранение)
 - **`requirements.txt`** — зависимости
 
 ## Установка
 
 Рекомендуется использовать виртуальное окружение.
 
 ```bash
 python3 -m venv .venv
 source .venv/bin/activate
 python3 -m pip install -U pip
 python3 -m pip install -r requirements.txt
 ```
 
 Примечание: в некоторых ячейках ноутбука может использоваться SMOTE. Если у тебя возникает ошибка импорта `imblearn`, установи дополнительно:
 
 ```bash
 python3 -m pip install imbalanced-learn
 ```
 
 ## Запуск
 
 1. Открой `sample.ipynb` в PyCharm/Jupyter.
 2. Убедись, что файл данных лежит по пути `data/dataset.xlsx`.
 3. Запускай ячейки сверху вниз.
 
 В ноутбуке формируется целевая переменная:
 
 - **`Рецидив да-1 нет -2`** — бинарная цель (0/1)
 
 ## Результаты
 
 В конце ноутбука формируется сравнительная таблица метрик для доступных (обученных) моделей со столбцами:
 
 - `Sensitivity`
 - `Specificity`
 - `Precision`
 - `Accuracy`
 - `F1-score`
 - `ROC AUC`
